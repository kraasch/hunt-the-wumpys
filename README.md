
# Hunt the Wumpys

A simple clone of the terminal game called [Hunt the Wumpus](https://www.atariarchives.org/bcc1/showpage.php?page=247) (1976).

There are some online versions, eg this one on [oscric.com](https://osric.com/wumpus/).
